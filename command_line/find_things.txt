Searching for files
------------------------------------------------------------------------------------------
\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\
------------------------------------------------------------------------------------------
locate				Find files by name
				Performs rapid database search of pathnames and then
				outputs every name that matches a given substring. locate
				uses a database created by the updatedb command
				periodically. If a recent file is not found in the db
				updatedb can be run as a superuser
------------------------------------------------------------------------------------------
find				Search for files in a directory hierarchy
------------------------------------------------------------------------------------------
xargs				Used to process the resulting list of files from a search
